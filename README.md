# 校园工具集（Campus Tools）

一款面向大学生的轻量网页工具集合，专注解决课程管理、考试规划、学习效率等高频需求，开源免费、无广告，支持多人协作贡献功能模块。

## 项目简介
- 定位：轻量化、模块化的校园效率工具平台
- 技术栈：HTML + CSS + JavaScript（无需复杂框架，本地可直接运行）
- 特色：开源可定制、多工具整合、支持同学提交功能贡献

## 核心功能列表
✅ 已实现功能：
- 课程表导出：支持手动输入课程信息，导出为PDF格式
- 考试倒计时：自定义考试日期，实时显示剩余天数

🔨 开发中功能：
- 笔记模板生成：提供Markdown/Word格式的课程笔记、实验报告模板
- 校园活动日历：聚合学校讲座、竞赛、活动信息，支持添加到个人日历

📌 待开发功能（欢迎提Issue/PR）：
- 图书馆座位预约提醒
- Excel课程表导入
- 二手交易信息聚合
